# You need to put the 'JJs_HashMonitor.ps1' and 'Run_HashMonitor.cmd' in the same folder as STAK-XMR.
# You also need to put 'Auto-Restarter+HashMonitor.bat' file, 'closePowershell' and 'closeXmrStak' Shortcuts in the same folder as STAK-XMR.
# You'll need to change properties of this 2 shortcuts to rune them as Administrator. For this:
# -> just right click on the shortcut, go in properties, click on Advanced and tick the box that say "Run as Administrator". Click apply and close.
# If you haven't already done yet or it's the first time you're using 'HashMonitor v2.7' by JerichoJones you also need to do this 2 more steps:
# -> Right click on the Widows startup button and click on "Windows PowerShell (Admin)" to run PowerShell as Administrator.
# -> Now just type the following and then press the Enter key: Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Bypass -Force
# CONGRATULATIONS !! You're now ready to run the Program !
# To do so, just run the 'Auto-Restarter+HashMonitor.bat' batch file.
